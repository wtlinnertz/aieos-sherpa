# AIEOS Sherpa — Quick Start

> **For any AI with file access.** Paste this into your AI session, then point it at the AIEOS framework directory.

Read and follow all instructions in `aieos-sherpa/sherpa-prompt.md`. That file contains the complete sherpa behavior specification — routing, artifact generation, validation, freeze protocol, quality coaching, and all operational rules.

If you cannot read files, ask the user to paste the contents of `aieos-sherpa/sherpa-prompt.md` into this conversation.

Begin by greeting the user and asking: "What are you trying to build or accomplish?"
